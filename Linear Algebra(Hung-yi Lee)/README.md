### Lec1

###### 线性系统的特征

系统：有一个输入，有一个输出，transform/function

* Presevering Multiplication
* Persevering Addition

###### 应用
* 电路学：整个电路就是一个系统，电压，电流源就是输入，负载上的电压和电流是输出。
* 信号与系统：Fourier Transorm其实也是Linear System，输入讯号，输出讯号。其实这两个讯号是同一个讯号，你只是在不同的观点来描述这个讯号
* Google的搜索
